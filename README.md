# 🎵 MIDI → Roblox Street Beats Converter

A Python tool that converts standard `.mid` files into playable beat scripts for the Roblox game **Street Beats**.

It parses MIDI tracks, extracts melody + bass information, quantizes timing, and generates a Lua payload compatible with the in-game ZAP beat system.

> Built for: [https://www.roblox.com/games/125756097576326/Street-Beats](https://www.roblox.com/games/125756097576326/Street-Beats)

---

## ⚡ Features

* 🎹 Auto-detect melody & bass tracks (Type 0, 1, 2 MIDI support)
* 🧠 Smart track filtering (removes drums / noise layers)
* 🎼 Quantized rhythm conversion (8 or 16 grid support)
* 🔊 Custom Roblox sound mapping (melody + bass samples)
* 🥁 Optional drum generation
* 🔀 Merge mode for full-song simplification
* 📊 Track listing / debugging mode
* ⚙️ Verbose conversion output

## 🚀 Usage

### 🔍 Inspect MIDI first

```bash
python midi.py --list-tracks song.mid
```

Shows:

* tracks / channels
* pitch range
* drum detection
* note density

---

### 🎧 Auto convert (recommended)

```bash
python midi.py song.mid
```

---

### 🎹 Choose specific tracks

```bash
python midi.py song.mid --melody-track 0 --bass-track 2
```

---

### 🎛 Type 0 MIDI (channel-based)

```bash
python midi.py song.mid --melody-channel 3 --bass-channel 1
```

---

### 🔥 Clean “melody-only” export

```bash
python midi.py song.mid --merge-tracks --no-drums
```

---

### 🎚 Advanced tuning

```bash
python midi.py song.mid \
  --bars 16 \
  --quantize 16 \
  --melody-sample 178 \
  --bass-sample 1
```

## 🎮 Output

The script generates a Lua file:

```lua
-- Auto-generated by midi.py
-- Made by Fxke
-- Source : song.mid
-- BPM    : 120
-- Bars   : 16
```

This file:

* builds a ZAP payload buffer
* sends it to Roblox via RemoteEvent
* plays as a structured beat sequence

---

## 🧠 How It Works

1. MIDI file is parsed using `mido`
2. Notes are grouped into bars
3. Timing is quantized to grid (8/16 cells)
4. Tracks are scored and filtered:

   * drums removed
   * pads reduced
   * melody prioritized
5. Roblox-compatible Lua payload is generated
6. Script fires data to `ReplicatedStorage.ZAP_RELIABLE`

---

## 🎯 Tips for Best Results

* Use **clean MIDI files (melody-focused works best)**
* Avoid orchestral MIDIs (too many layers)
* For best accuracy:

  * pick highest-density melodic track manually
  * use `--list-tracks` first
* If sound is messy → try `--merge-tracks`

---

## ⚠️ Notes

* Not all MIDIs convert perfectly (layered orchestral files will lose clarity)
* Some Roblox instruments may sound different depending on your sound pack
* Game updates may break remote format compatibility
